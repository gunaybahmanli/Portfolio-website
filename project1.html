Project 1: Electronic Sales Data Analysis

Short description: Analyzing customer purchase behavior to extract insights and improve sales strategies.

Introduction

This project focuses on analyzing electronic sales transactions from September 2023 to September 2024. The goal is to identify patterns in customer behavior, product sales performance, and revenue distribution.

Objective

The main objective is to provide actionable insights to improve marketing strategies, identify top-selling products, and segment customers based on their purchasing behavior.

Data & Tools

  • Dataset: Customer purchase behavior - Electronic Sales Data
  • Tools: Python, Pandas, Matplotlib, Seaborn
  • SQL for data extraction
  • Power BI for visualization (optional)

Methodology

Data cleaning, feature engineering, and exploratory data analysis (EDA) were conducted to prepare the dataset. Key metrics such as total sales, purchase frequency, and loyalty changes were calculated.

SQL / Python Scripts

Example of Python code used for analysis:

import pandas as pd

data = pd.read_csv("electronic_sales.csv")
summary = data.groupby('Product Type')['Total Price'].sum()
print(summary)

Visualizations

Charts show sales distribution, top products, and customer behavior insights.

Insights & Conclusion

  • Top products contributed to 40% of total revenue.
  • Loyalty members had higher purchase frequency than non-members.
  • Customers aged 25-35 were the most active demographic.
  • SQL and Python analysis helped identify sales trends and potential improvements.
