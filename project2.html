Project 2: Employee Performance Dashboard

Short description: Building a comprehensive dashboard to analyze employee performance, productivity, and HR metrics.

Introduction

This project focuses on analyzing employee data including performance scores, satisfaction, and turnover rates. The goal is to provide actionable insights for HR and management to improve productivity and retention.

Objective

The main objective is to create a dashboard that highlights key performance indicators (KPIs), identifies trends, and helps management make informed decisions regarding workforce management.

Data & Tools

  • Dataset: Employee performance and HR data
  • Tools: Python, Pandas, Matplotlib, Seaborn
  • SQL for data extraction
  • Power BI for visualization

Methodology

Data cleaning, feature engineering, and exploratory data analysis (EDA) were performed. Key metrics such as average performance score, overtime hours, and resignation rate were analyzed.

SQL / Python Scripts

Example of Python code used for analysis:

import pandas as pd

data = pd.read_csv("employee_data.csv")
avg_perf = data.groupby('Department')['Avg_Performance_Score'].mean()
print(avg_perf)

Visualizations

Charts show employee performance trends, departmental comparisons, and turnover analysis.

Insights & Conclusion

  • Departments with higher training hours had better average performance scores.
  • High overtime correlated with lower satisfaction scores.
  • Resignation rates were higher in departments with low average performance.
  • SQL and Python analysis helped management prioritize HR interventions.
